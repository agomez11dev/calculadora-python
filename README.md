# 🧮 Calculadora AG

Mi primera calculadora hecha en Python.

## Funciones actuales

* Suma
* Resta
* Multiplicación
* División
* Promedio de notas

## Sobre el proyecto

Hice este proyecto para practicar Python y mejorar mi lógica de programación.
La idea es seguir mejorándolo con el tiempo mientras aprendo más.

## Cómo ejecutarlo

Abrir la terminal en la carpeta del proyecto y ejecutar:

```bash
python calculadora.py
```
